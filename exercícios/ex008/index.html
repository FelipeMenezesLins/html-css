Principais formatações

Negrito/italico

Nesta frase iremos deixar a frase em negrito usando a tag B

Nesta frase iremos deixar a frase em negrito de maneira semântica usando a tag STRONG

Italico/ênfase

Nesta frase iremos deixar a frase em italico usando a tag I

Nesta frase iremos deixar a frase em italico (semântica) usando a tag "em"

Texto Marcado

Para deixar o texto marcado você usa a tag mark

E para marca o texto com outra cor usamos a parâmetro

Texto grande e texto pequeno

Agora iremos deixar o texto grande e pequeno usamos a tag big e small

Texto excluido

Marcamos isso para o texto ser lido mas não considerado nós usamos a tag del

Texto inserido

Podemos marca o como inserido para mostra ênfase ou que foi inserido depois usamos a tag ins

Também podemos utilizar de outra forma usando a tag u de maneira não(semântica)

Texto sobrescrito

Usando o exemplo x20+3 sem a tag e com a tag sup x20+3

Texto subescrito

Usando como exemplo h20 sem a tag e com a tag sub h20
